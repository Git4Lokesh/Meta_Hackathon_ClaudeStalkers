# LoRA weights and tokenizer blobs

This folder in **git** only stores metrics, plots, and small config files so pushes stay under GitHub’s file-size limits.

Download the full adapter (including `adapter_model.safetensors` and tokenizer files) from Hugging Face:

```bash
hf download GeminiHugger/war-room-grpo-multirole-v2 --local-dir outputs/war_room_grpo_multirole_v2
```
